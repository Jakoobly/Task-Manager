# 📝 Task Manager

A simple Java console application to manage tasks.  
Built to practice object-oriented programming concepts.

---

## ✨ Features

- Create tasks  
- Edit tasks  
- Delete tasks  
- Show all tasks  
- Filter open tasks  
- Set priority (LOW, MEDIUM, HIGH)  
- Save tasks to file  

---

## 🛠️ Technologies

- Java  
- IntelliJ IDEA  
- Git
